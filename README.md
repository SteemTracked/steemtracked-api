# SteemTracked API
Connects to SteemSQL (MSSQL Database) to get data for https://steemtracked.com

## Installation

```bash
git clone https://github.com/steemtracked/steemtracked-api.git
cd steemtracked-api

npm install
```

## Run Development

(Port set to 3001 and can be changed in the server.js file)
```bash
npm run start-dev
```

## Run Production

```bash
npm start
```
or
```bash
node server.js
```
